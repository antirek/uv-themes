# uv-themes

1. набор онлайн-утилит для разработчиков организации
   
обоснование: многие онлайн-утилиты становятся трудноступными (блокировки cdn, замедление)

решение: быстроразворачиваемый комплекс онлайн-утилит в периметре организации

исполнение: репо, набор утилит, докер-контейнер

подробнее https://github.com/antirek/tools

   
2. быстрое построение закрытой корпоративной чат-системы с помощью ИИ-моделей

обоснование: требуется создавать чат-системы в организации для внутренних сервисов или обслуживания клиентов

решение: разработка решения с помощью ИИ-моделей из готовых компонент по имеющимся спецификациям

исполнение: репо, набор спек для компонент, пример спеки для создания чат-системы

https://github.com/chottodev/mms3
https://github.com/mobilon-dev/chotto


3. онлайн база знаний для сотрудников и ИИ (RAG)

обоснование: в организации много репозиториев с кодом, с запущенными инсталляциями, по которым требуется быстрый поиск информации, решение вопросов

решение: регулярно собирать данные в одну базу данные (из репо), индексировать в вектор бд, искать данные человеком или ИИ

исполнение: репо, скрипт краулера, бд qdrant?, интерфейс чата (RAG), MCP 

https://habr.com/ru/articles/1029740/
